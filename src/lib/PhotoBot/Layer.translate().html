Syntax

canvas.layers[i].translate(x, y)

Description

Positions layer i at the given coordinates. The x and y parameters define where to position the top left corner of the layer, measured from the top left of the canvas, in pixels.

Example

photobot = ximport("photobot")
canvas = photobot.canvas(100,100)
canvas.layer("robot.jpg")
for i in range(70):
  canvas.layers[1].duplicate()
  canvas.layers[2+i].translate(0,1+i)
