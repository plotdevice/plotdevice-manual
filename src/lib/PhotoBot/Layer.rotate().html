Syntax

canvas.layers[i).rotate(angle]

Description

Rotates layer i. Rotates the layer by given angle (1 to 360). Positive numbers rotate counter-clockwise, negative numbers rotate clockwise. Rotate commands are executed instantly, so many subsequent rotates will distort the image.

The rotate command is off sometimes by a single pixel - this will be fixed in later versions of PhotoBot.

Example

photobot = ximport("photobot")
canvas = photobot.canvas(100,100)
canvas.layer("robot.jpg")
canvas.layers[1].opacity(40)
for i in range(10):
  canvas.layers[1].duplicate()
  canvas.layers[2+i].rotate(10*i)
