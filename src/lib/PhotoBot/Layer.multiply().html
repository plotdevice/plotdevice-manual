Syntax

canvas.layers[i].multiply()

Description

Sets the blend mode of layer i to multiply. Multiplying an underlying image with black results in black, multiplying with white leaves the image unchanged. So multiply usually returns a darker result.

Example

photobot = ximport("photobot")
canvas = photobot.canvas(100,100)
canvas.layer("robot.jpg")
i = canvas.layer("newspaper.jpg")
canvas.layers[i].multiply()
