Syntax

canvas.layers[i].sharpness(value=1.0)

Description

Increases or decreases the sharpness in layer i. The given value is a percentage to increase or decrease the image contrast, for example 1.2 means sharpness at 120%.

Example

photobot = ximport("photobot")
canvas = photobot.canvas(100,100)
canvas.layer("robot.jpg")
canvas.layers[1].sharpness(1.5)
