measure()

Syntax

measure(grob)
measure("text", width=None, height=None, **fontstyle)

Description

Returns width & height for graphics objects, strings, or images. If the first argument is a string, the size will reflect the current font() settings and will layout the text using the optional width and height arguments (as well as any keyword arguments supported by the text() command).

If called with a file object, PlotDevice will expect it to be an image file and return its pixel dimensions.

If called with a Bezier, returns its "bounding box" dimensions.

Returns

a Point tuple with the object's dimensions
