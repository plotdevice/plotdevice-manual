choice()

Syntax

choice(sequence)

Description

Returns one random item from the sequence parameter (a tuple, list, or string). The choice() command is syntactic sugar for the comparatively awkward lst[random(len(lst)] command.

Returns

a random item from the given sequence

Tutorial

Lists

Example

words = ["It is certain", "Reply hazy try again", "Outlook not so good"]
c = choice(words)
# Variable c contains a random item from the words list
# It might now be "It is certain"...
# or "Reply hazy try again"...
# or "Outlook not so good"
