autotext()

Syntax

autotext()

Description

Generates random text from an XML-document. Autotext is a very powerful feature once you learn how to use it. PlotDevice has a built-in implementation of the Kant Generator Pro. XML-files for the Kant Generator have a specified structure. Further information can be found at Dive into Python's chapter on XML Processing.

Returns

a string containing automatically generated text

Tutorial

Strings

Example

fontsize(9)
lineheight(0.8)
txt = autotext("kant.xml")
text(txt, 5, 10, width=110)
