Point

Syntax

Point(x, y)

Description

A named tuple representing an x/y coordinate. You can access its properties by name in addition to using the object as a ‘normal’ tuple. Point objects also provide utility methods for geometry calculations.

Properties

pt.x
pt.y

Methods

pt.angle(point) # or (x, y)
pt.distance(point) # or (x, y)
pt.reflect(point, d=1.0, a=180) # or (x, y, d, a)
pt.coordinates(distance, angle)

See the tutorial on Point Geometry to learn more about Point objects and their uses.
