textpath()

Syntax

textpath(txt, x, y, width=None, height=1000000)

Description

Works exactly like the text() command, except it returns a path that can be used with bezier() or clip() instead of drawing text to the screen.

Returns

BezierPath containing the given text as outlines

Tutorial

Paths, Strings

Example

fontsize(65)
path = textpath("clip", 10, 70)
with clip(path):
    image("header.jpg", -300, -150)
