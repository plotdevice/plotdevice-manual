textwidth()

Syntax

textwidth(txt, width=None)

Description

Returns what would be the width of a text block drawn with the text() command, in the current font(), fontsize() and lineheight(). The parameter sets the text string to measure.

Returns

the width of the text block, as a floating point number

PlotDevice Equivalent

The measure() command will return a (width, height) tuple when called with a string or a Text object as its first argument.

Example

font("Courier New")
w = textwidth("hello")
print w
