textheight()

Syntax

textheight(txt, width=None, **options)

Description

Returns what would be the height of a text block drawn with the text() command given the current font() and layout() settings. The first parameter sets the text string to measure. The optional second parameter sets a word-wrap margin and measures the resulting text block. The textheight() command also accepts the same optional keyword arguments as the text() command.

Returns

the height of the text block, as a floating point number

PlotDevice Equivalent

The measure() command will return a Size when called with a string or a Text object as its first argument.

Example

font("Courier New")
h = textheight("hello", 30)
print(h)
