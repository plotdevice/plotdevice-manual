align()

Syntax

align(style=LEFT)

Description

Sets the alignment used by the text() command. Valid style arguments are LEFT, RIGHT, CENTER, or JUSTIFY. When calling text() without specifying a width argument, its x & y coords will determine the left-, right-, or center-point of the line of text (based on the alignment).

Calling text() with a width argument will lay out a column of type. Text in a column wraps across multiple lines within the rectangle defined by the x/y/w/h dimensions and the align() setting controls how lines are broken within this block.

Returns

the current alignment

Example

font(12)
x = 62
line(x,10, x,115, dash=3, stroke=.7)
align(RIGHT)
text("Conapt", 62,25)
align(LEFT)
text("Kipple", 62,65)
align(CENTER)
text("Homeopape", 62,105)

quip = "Twas brillig and the slithy toves"
x,y = 10,55
font(16)
align(RIGHT)
text(quip, x,y, width=90)
with stroke(.7), nofill():
    rect(x,y-font().size, 90, 70, dash=3)
