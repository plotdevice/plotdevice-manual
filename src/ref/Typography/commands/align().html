align()

Syntax

align(align=LEFT)

Description

Sets the alignment of text in a block. Text in a block wraps across multiple lines when the width parameter of the text() command is specified. The parameter for align() can be either LEFT, RIGHT, CENTER, JUSTIFY.

Returns

the current alignment

Example

fill(0.2)
fontsize(16)
align(RIGHT)
quip = "If it ain't fun make it fun"
text(quip, 10, 55, width=90)
