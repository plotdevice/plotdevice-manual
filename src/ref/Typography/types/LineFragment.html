LineFragment

Description

The Text, TextMatch, and TextFrame objects all provide a property called lines which returns a list of one or more LineFragment objects. Each LineFragment includes character and layout information for a single line's-worth of text.

Typically, a LineFragment describes the full set of characters visible on a given line, but in cases where you're examining a TextMatch, the layout and baseline properties will only apply to the matched text, not any characters before or after it on the line.

Geometry Properties (read-only)

fmt.bounds # a Region with the full size of the line
fmt.layout # the Region within the line bounds occupied by characters
fmt.baseline # the origin Point of the first character in the line

Text Properties (read-only)

fmt.span # a tuple with the start and end indices for characters in the fragment
fmt.text # a string representation of the line's characters
fmt.frame # the TextFrame object in which the line is contained

Example

font('avenir', 32, leading=1.6)
t = text(20,60, 230, str='Blixa Bargeld, poet laureate')

nofill()
for slug in t.lines:
    rect(slug.bounds, stroke=.9)
    rect(slug.layout, stroke=.6)
    arc(slug.baseline, 4, fill='red')
