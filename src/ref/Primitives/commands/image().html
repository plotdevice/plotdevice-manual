image()

Syntax

image(path, x, y, width=None, height=None, data=None, plot=True, **style)

Description

Draws an image to the screen. An image is a JPEG-, GIF-, PNG-, AI-, EPS-, TIF-, PSD-, or PDF-file stored on disk. For vector works, EPS is the best format to use. The first parameter sets the pathname of the image file. By default, PlotDevice looks for files in the folder where the PlotDevice file is saved. The following two parameters set the location of the image.

The width and height parameters are optional and define the maximum width and height of the image. When set, the image is scaled to fit within the given width and height while maintaining its aspect ratio. The alpha and blend ‘style’ parameters are optional as well, and set the opacity and blend mode for compositing the image.

The optional data parameter is used when supplying an image as a data stream instead of a file path. As with other primitives, setting the plot parameter to False will return a drawable object but not add it to the canvas.

Returns

an Image object that can be passed to background(), fill(), clip(), and mask() among others.

Example

image("superfolia.jpg", 50, 0)
