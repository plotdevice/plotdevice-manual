poly()

Syntax

poly(self, x, y, radius, sides=3, plot=True, **style)

Description

Draws an n-sided regular polygon (by default a triangle) centered at (x,y) and sized according to radius. The sides argument sets the type of polygon to draw. Absent any rotate() transformations, the polygon will be oriented such that its base is horizontal.

Returns

Bezier containing the rectangle

Tutorial

Primitives, Paths

Example

fill(0.2)
poly(50,50, 40)
