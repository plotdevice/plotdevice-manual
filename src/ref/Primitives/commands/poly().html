poly()

Syntax

poly(self, x, y, radius, sides=4, points=None, plot=True, **style)

Description

Draws an n-sided convex polygon (by default a square) centered at (x,y) and sized according to radius. The sides argument sets the type of polygon to draw. Absent any rotate() transformations, the shape will be oriented such that its base is horizontal.

If you include a points argument (in lieu of sides), a concave "star" polygon with regularized angles will be drawn instead.

Returns

Bezier containing the rectangle

Tutorial

Primitives, Paths

Example

fill(.2)
poly(30,30, 20)
poly(80,30, 20, sides=5)
poly(30,80, 20, sides=6)
poly(80,80, 20, sides=8)
fill(.2)
poly(30,30, 20, points=5)
poly(80,30, 20, points=6)
poly(30,80, 20, points=8)
poly(80,80, 20, points=12)
