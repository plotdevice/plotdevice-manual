oval()

Syntax

oval(x, y, width, height, plot=True, **style)

Description

Draws an ellipse to the screen. The x and y parameters set the location of the oval's top-left ‘corner’. The third parameter sets the width, and the fourth sets the height (i.e., setting them to the same value will yield a circle).

Returns

a Bezier object containing the oval

Tutorial

Primitives, Paths

Example

fill(0.2)
oval(10,20, 40,40)
