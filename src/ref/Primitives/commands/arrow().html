arrow()

Syntax

arrow(x, y, width, type=NORMAL, draw=True)

Description

Draws an arrow to the screen. The first two parameters set the location (measured from the arrowâ€™s head), the third sets the width. Optionally you can define the arrow type with the fourth parameter. By default the arrow type is NORMAL, but there is also a trendier FORTYFIVE version.

Returns

BezierPath containing the arrow

Primitives, Paths

Example

fill(0.2)
arrow(50, 50, 50)
