line()

Syntax

line(x1, y1, x2, y2, draw=True)

Description

Draws a line to the screen. A line is a straight path between two points. The first two parameters set the location of the first point (origin), the following two parameters set the location of the second point (destination). You will not notice a line if it’s stroke() is the same as the background color, or if the line’s strokewidth() is zero.

Returns

BezierPath containing the line

Tutorial

Primitives, Paths

Example

stroke(0.2)
strokewidth(2)
line(10, 20, 80, 80)
