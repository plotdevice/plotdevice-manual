line()

Syntax

line(x1, y1, x2, y2, plot=True)

Description

Draws a line-segment to the screen â€“ a straight path between two points. The first pair of parameters sets the location of the "origin" point and the following pair sets the location of the "destination" point.

You will only be able to see the line segment if the stroke() is different from the current background() color. Also note that lines will not be drawn if the stroke() is set to None.

Returns

a Bezier object containing the line

Example

pen(2)
stroke(0.2)
line(10, 20, 80, 80)
