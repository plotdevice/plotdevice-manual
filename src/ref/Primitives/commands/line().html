line()

Syntax

line(x1, y1, x2, y2, plot=True)

Description

Draws a line to the screen. A line is a straight path between two points. The first two parameters set the location of the first point (origin), the following two parameters set the location of the second point (destination). You will not notice a line if its stroke() is None or the same as the background color (or if the pen()'s nib is zero).

Returns

a Bezier object containing the line

Tutorial

Primitives, Paths

Example

pen(2)
stroke(0.2)
line(10, 20, 80, 80)
