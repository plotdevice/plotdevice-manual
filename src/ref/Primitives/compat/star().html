star()

Syntax

star(x, y, points=20, outer=100, inner=50, plot=True, **style)

Description

Draws a star to the screen. The x and y parameters set the location of the starâ€™s center. There are three optional parameters that set the number of points, the outer radius and the inner radius (the fill radius).

If the inner radius is omitted, one will be chosen such that the resulting star has regularized angles.

Returns

a Bezier object containing the star

Tutorial

Primitives, Paths

Example

fill(0.2)
star(50, 50, points=5, outer=40)
