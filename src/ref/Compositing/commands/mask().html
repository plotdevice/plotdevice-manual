mask()

Syntax

with mask(path):

Description

Using the clip() command defines a clipping mask. The supplied parameter defines the path to be used as a clipping mask. Each of the shape commands returns a path that can be used with clip() â€“ setting the draw parameter of a shape command to False simply returns the path without actually drawing the shape. Any shapes, paths, texts and images drawn inside the clip() block are clipped, this means that any part that falls outside the clipping mask path is not drawn. After completing the block, subsequent shapes, paths, texts and images are no longer clipped to the path supplied to clip().
