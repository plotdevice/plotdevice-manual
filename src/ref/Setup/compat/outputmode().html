outputmode()

Syntax

outputmode(mode)

Description

Changes the way colors are displayed. While the colormode() command specifies the input of colors, outpumode() specifies the output. By default, the output mode for colors is RGB, but you can also set it to CMYK so that exported PDFâ€™s are ready for print. Output in RGB will look brighter on screen but to validate PDF documents the colors need to be in CMYK.

Returns

the current output mode (RGB or CMYK)
