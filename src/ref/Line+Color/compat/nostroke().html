nostroke()

Syntax

nostroke()

Description

Disables the current stroke color. All shapes, paths and text following this command will be drawn without a stroke (outline) color.

Tutorial

Color

Example

fill(0.2)
strokewidth(6)
stroke(1.0, 0.0, 0.5)
rect(10, 10, 20, 40)
nostroke()
rect(40, 10, 20, 40)
