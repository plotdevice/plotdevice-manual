nofill()

Syntax

nofill()

Description

Disables the current fill color. All shapes, paths and text following this command will be drawn without a fill color. Calling nofill() is equivalent to fill(None).

Tutorial

Color

Example

strokewidth(1.5)
stroke(0.2)
fill(0.2)
rect(10, 10, 20, 40)
nofill()
rect(40, 10, 20, 40)
