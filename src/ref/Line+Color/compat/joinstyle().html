joinstyle()

Syntax

joinstyle(style)

Description

Defines how vertices in a path should be rendered. The style argument must be one of: MITER (the default), ROUND, or BEVEL.

Tutorial

Paths

Example

fill(None)
stroke(0)
strokewidth(10)

joinstyle(MITER)
bezier([(20,20), (40,40), (60,20)], curvature=0)

joinstyle(ROUND)
bezier([(20,50), (40,70), (60,50)], curvature=0)

joinstyle(BEVEL)
bezier([(20,80), (40,100), (60,80)], curvature=0)
