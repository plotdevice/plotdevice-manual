strokewidth()

Syntax

strokewidth(width)

Description

Sets the width of the stroke used for lines around shapes, paths and text.

Returns

the current stroke width

PlotDevice Equivalent

The pen() command allows you to set the stroke width using its nib keyword argument or simply by passing the new width as its first positional arg.

Example

nofill()
stroke(0.2)
strokewidth(1.5)
rect(10, 10, 20, 40)
strokewidth(3)
rect(40, 10, 20, 40)
