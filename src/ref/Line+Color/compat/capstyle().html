capstyle()

Syntax

capstyle(style)

Description

Defines how endpoints of a path should be rendered. The style argument must be one of: BUTT (the default), ROUND, or SQUARE. Note that the latter two styles extend the length of the path to add the endcaps.

Tutorial

Paths

Example

fill(None)
stroke(0)
strokewidth(10)
capstyle(BUTT)
line(20,20, 50,20)
capstyle(ROUND)
line(20,40, 50,40)
capstyle(SQUARE)
line(20,60, 50,60)
