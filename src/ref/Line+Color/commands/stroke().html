stroke()

Syntax

stroke(r, g, b, a)
stroke(h, s, b, a)
stroke(c, m, y, k, a)
stroke(k, a)
stroke(color)

Description

Arguments will be interpeted according to the canvas's current color-mode and range. For details see the color() command.

Sets the current stroke color. The stroke is used to draw lines around shapes, paths and text. All drawing operations following this command will be outlined in this stroke color.

The color is specified in terms of RGB, HSB or CMYK depending on the canvas's current color-mode and range. For details see the color() command. To disable stroke-drawing either call stroke() with None or use the nostroke() convenience method.

Returns

Color containing the defined color values

Example

nofill()
strokewidth(3)
