color()

Syntax

color(k, a=1)          # greyscale
color(r, g, b, a=1)    # RGB mode
color(h, s, b, a=1)    # HSB mode
color(c, m, y, k, a=1) # CMYK mode

color(HSB)             # have color commands expect HSB values
color(range=255)       # use 0-255 component values rather than 0–1
color(CMYK, range=100) # set both the current mode and range

Description

Returns a color for use with the fill() , stroke(), and font() commands. The color is specified in terms of RGB (the default), HSB, or CMYK components depending on the current color mode. Each component value is interpreted based on the current color range (by default 0–1.0). The mode and range can be passed as parameters if you wish to e.g., create a CMYK color even though the current mode is RGB.

When called with just a mode or range parameter (and no component values), the color() command will update the canvas's ‘current’ value for that parameter. Subsequent calls to color(), stroke(), or fill() can then use the specified mode or range without including the parameters in every call.

Returns

Color containing the defined color values

Example

clr1 = color(1.0, 0.0, 0.5)
fill(clr1)
rect(10, 10, 25, 25)
