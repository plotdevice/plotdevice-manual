arcto()

Syntax

arcto(x, y, cx, cy, radius, close=False)

Description

Draws a circular arc at the corner of the triangle formed by the current point in the active path, a destination point, and a ‘control’ point at the peak of the arc. Be sure to specify a radius that is small enough to fit within that angle. If the close argument is set to True, a lineto() command will be synthesized connecting the end of the arc to the origin point of the active subpath.

The arcto() command must be called inside a bezier() context (or between beginpath() and endpath()).

Example

with bezier(30, 50, stroke=0.2, fill=None, close=False):
    arcto(80,50, 55,100, 10)
or 
nofill()
