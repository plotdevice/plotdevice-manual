moveto()

Syntax

moveto(x, y)

Description

Moves the ‘current’ point of the path to a new location (x,y) without drawing a line from the previous point. This is conceptually similar to lifting your pen from the page and putting it down somewhere else to continue drawing.

The moveto() command must be called inside a bezier() context (or between beginpath() and endpath()).

Tutorial

Paths

Example

with bezier(10, 10, stroke=0.2) as path:
    lineto(50, 100)
    moveto(60, 100)
    lineto(100, 100)
or 
stroke(0.2)
beginpath(10, 10)
lineto(50, 100)
moveto(60, 100)
lineto(100, 100)
endpath()
