drawpath()

Syntax

drawpath(path)

Description

Draws a path to the screen. A path is a series of lines and curves defined between beginpath() and endpath(). Normally, endpath() draws the path to the screen, unless when calling endpath(plot=False). The path can then be assigned to a variable, and this variable used as a parameter for drawpath().

Note: if you have one path that you want to draw multiple times with drawpath(), for example each with its own rotation and position, you need to supply a copy:
drawpath(path.copy())

PlotDevice Equivalent

The new plot() command will draw Bezier objects (as well as images and text) to the canvas.

Tutorial

Paths

Example

stroke(0.2)
beginpath(10, 10)
lineto(40, 10)
p = endpath(plot=False)
drawpath(p)
