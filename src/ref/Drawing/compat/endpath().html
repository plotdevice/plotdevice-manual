endpath()

Syntax

endpath(draw=True)

Description

The endpath() command is the companion to beginpath() and may only be called after the later. When endpath() is called, the path defined between beginpath() and endpath() is drawn to the screen. Optionally, when endpath(draw=False) is called, the path is not drawn to the screen, but can be assigned to a variable and drawn to the screen at a later time with the drawpath() command.

Returns

a Bezier object containing the constructed path

PlotDevice Equivalent

The endpath() command has been subsumed into the new bezier() command. Rather than calling beginpath() and endpath() to construct beziers, you can call bezier() as part of a with statement.

Example
