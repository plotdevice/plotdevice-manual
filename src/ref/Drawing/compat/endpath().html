endpath()

Syntax

endpath(draw=True)

Description

The endpath() command is the companion to beginpath() and may only be called after the later. When endpath() is called, the path defined between beginpath() and endpath() is drawn to the screen. Optionally, when endpath(plot=False) is called, the path is not drawn to the screen, but can be assigned to a variable and drawn to the screen at a later time with the drawpath() command. Check the tutorial on paths for other possibilities with paths.

Returns

a Bezier object containing the constructed path

Tutorial

Paths

Example

stroke(0.2)
beginpath(10, 10)
lineto(40, 10)
endpath()

beginpath(10, 10)
lineto(40, 10)
p = endpath(plot=False)
drawpath(p)
