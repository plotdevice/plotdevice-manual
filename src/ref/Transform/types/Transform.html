Transform

Syntax

Transform()

Description

A personal transformation state you can use to apply translation, rotation, scaling and skewing to a path before drawing it.

Methods

t.translate(x=0, y=0)
t.rotate(degrees=0, radians=0)
t.scale(x=1, y=None)
t.skew(x=0, y=0)
t.invert()

t.append(t)
t.prepend(t)

t.apply(path_or_point)

t.copy()

See the tutorial on paths to learn more about the methods.

The apply() method returns a new BezierPath or Point with the defined transformations applied to it.
