scale()

Syntax

scale(x, y=None)

Description

Increases, decreases or streches the size of all subsequent shape, path, text, image commands. The first parameter sets the horizontal scale, the second parameter the vertical scale. Additionally, the scale() command can be called with a single parameter that sets both the horizontal and vertical scale.

Scale values are specified as floating-point (decimal) numbers. The scale() command works incrementally: if you call scale(0.5), and later on call scale(0.2), all subsequent drawing commands will be sized to 10% (0.2 of 0.5).

Context Manager

As with the other transform commands, scale() can be used in a with statement to limit its scope to drawing commands within the following block of code.
