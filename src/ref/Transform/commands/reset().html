reset()

Syntax

reset()

Description

Resets all transformations to their default values. PlotDevice is a so-called state-machine. For example, once rotate(90) is called, all subsequent shapes, paths, text and images are rotated 90 degrees. The rotation state is 90 degrees. If rotate() is called again, for example rotate(30), the rotation state becomes 120 degrees (90+30). Once reset() is called, the rotation state becomes 0 degrees (the default rotation) again.
