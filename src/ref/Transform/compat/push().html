push() & pop()

Syntax

push()

pop()

Description

PlotDevice is a so-called state-machine. For example, once rotate(45) is called, all subsequent shapes, paths, text and images are rotated 45 degrees – the rotation ‘state’ has been set to 45°. While the transform() command provides a mechanism for controlling the scope of these state changes within blocks, the push() & pop() commands allow your script to save and restore the state manually.

All transformations, such as rotate() and skew(), defined between a push() and pop() call last only until pop() is called. Thereafter the previous state is used. For example, a rotate(-30) within a rotate(45) means a rotation state of 15 degrees until pop() is called and the rotation state becomes 45 degrees again.

Example
