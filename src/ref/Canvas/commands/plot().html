plot()

Syntax

plot(grob)

Description

The plot() command will draw any PlotDevice primitive to the canvas. It's useful in combination with the plot argument that can be passed to rect(), oval(), text(), image(), and friends.

When you call a drawing-related function with plot=False, the object is created but not added to the canvas. You can save a reference to the object in a variable then pass it to plot() later to be drawn as-is or with additional styling in its keyword args.

The plot command can also be used as part of a with statement to set whether drawing commands affect the canvas by default. Using with plot(False): will disable drawing within the block (allowing you to omit the plot argument in the individual commands

Tutorial

Graphics State

Example

# create a shape (but don't draw it immediately)
r = rect(20,20,40,40, plot=False)
...
# draw the saved shape (but override the canvas's fill color)
plot(r, fill='red')

# the plot keyword arg prevents this from being drawn
o = oval(0,0,100,100, plot=False)

# the plot() command disables drawing for the entire block
with plot(False):
    o = oval(0,0,100,100)   # not drawn
    s = rect(100,100,10,10) # same here
