clear()

Syntax

clear()       # erase the canvas
clear(all)    # erase the canvas and reset drawing state
clear(*grobs) # remove specific objects from the canvas

Description

Erases any prior drawing to the canvas and can optionally reset the graphics state (transform, colors, and compositing) when called with the all argument. Calling clear() with one or more references to previously-drawn objects will remove just those objects without otherwise affecting the drawing state.

Graphics State

Example

r = rect(0,0, 100,10) # add a rectangle
t = poly(50,50, 25)   # add a square
c = arc(125,125, 50)  # add a circle
clear(r, c) # remove the rectangle & circle
