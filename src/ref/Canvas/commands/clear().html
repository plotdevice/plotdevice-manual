clear()

Syntax

clear(canvas, transform, plotstyle) # reset everything to factory defaults
clear(all) # same as above
clear(canvas) # erases the canvas but leaves transforms and plotstyle alone

Description

The clear() command erases any prior drawing to the canvas and can optionally reset the transform and plotstyle as well.
