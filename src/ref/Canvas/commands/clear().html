clear()

Syntax

clear()       # erase the canvas
clear(all)    # erase the canvas and reset drawing state
clear(*grobs) # remove specific objects from the canvas

Description

The clear() command erases any prior drawing to the canvas and can optionally reset the transform, colors, and compositing state (when called with all). Passing references to previously-drawn objects will remove just those objects without affecting the drawing state.
